# SafetyNet Alert System

## Description
SafetyNet Alerts is an application designed to provide emergency responders with critical information during urgent situations like fires, storms, and floods. It processes and stores data such as phone numbers, medical conditions, and addresses to help responders quickly locate and assist individuals in need. Built with Spring Boot, the application offers RESTful endpoints for real-time access to health status, metrics, and location data, while also allowing updates to resident information. The system follows the Model-View-Controller (MVC) pattern, ensuring scalability, maintainability, and reliability.
